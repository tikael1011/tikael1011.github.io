IndoorNav

Main language: Java, Python

This research proejct simulates the following (similar) situation: A gunman shows up inside a building, and people start to evacuate. They open our app, then they know where they are and will receive evacuation routes based on their current location. There are four main parts of this project: wifi signal strength based localization, route planning, image processing and SDN (software defined network).
My work is mainly on wifi part and I am responsible for integration of the whole program.
One of the challenge I met is that the signal strength varies along time and other factors. The second is that utilizing signal strength alone is not enough to achieve high accuracy (error < 1m). The third is that while people using our app, they move fast. We should try to minimalize the delay both in communication between app and server and localization/route computation.

Questioning and Answering System

Main language: Java

This is a co-op research project with UMMS, IBM and WPI. Mainly purpose is to determine whether an article is friendly to people with CD. How do we measure that? We achieve this by calculating the correlation between "Rougue System rating" and manual rating. If the correlation is high, then we can use program instead of human, which is faster and more ojective.
